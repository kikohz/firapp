## Requirements
- Vapor 4
- Swift 5+

Mac or Liunx (swift environment installed)
[Deploying](https://docs.vapor.codes/4.0/deploy/digital-ocean/)



## Usage

Install the vapor environment, clone the project, and execute vapor run
